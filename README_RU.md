**[myip.xin](https://myip.xin)** — это быстрый, надежный и удобный инструмент для разработчиков и пользователей, позволяющий мгновенно узнать свой публичный IP-адрес (IPv4/IPv6), проверить геолокацию IP, запустить тест утечки DNS и просмотреть WHOIS-данные. 

Интерфейс оптимизирован как для веб-браузеров, так и для консольных утилит командной строки (CLI). Сервис работает на глобальной сети Cloudflare, гарантируя высокую скорость и полную конфиденциальность без отслеживания пользователей.

---

## 🚀 Основные возможности

*   **Быстрая проверка IP**: Мгновенно показывает ваш текущий внешний IP-адрес.
*   **Поддержка CLI**: Получайте IP-адрес или геолокацию прямо в терминале с помощью `curl`, `wget` или `httpie`.
*   **Геолокация по IP**: Полные географические данные, включая город, регион, страну, координаты (широта/долгота), почтовый индекс, провайдера (ASN) и часовой пояс.
*   **Совместимый JSON**: Эндпоинт `/json` возвращает данные в формате, полностью идентичном `ipinfo.io/json`, что позволяет легко заменить старый API в готовых скриптах.
*   **Конфиденциальность**: Без логов и трекеров на сервере.

## 💻 Использование в командной строке

Вы можете легко отправлять автоматические запросы к **myip.xin** из терминала.

### 1. Получить IP-адрес в виде чистого текста

Удобно для скриптов автоматизации:

```bash
curl https://myip.xin
```

**Пример вывода:**
```text
57.182.85.71
```

### 2. Получить геолокацию в формате JSON (совместимо с ipinfo.io)

Возвращает детальную информацию в формате JSON:

```bash
curl https://myip.xin/json
```

**Пример вывода:**
```json
{
  "ip": "57.182.85.71",
  "hostname": "ec2-57-182-85-71.ap-northeast-1.compute.amazonaws.com",
  "city": "Tokyo",
  "region": "Tokyo",
  "country": "JP",
  "loc": "35.6895,139.6917",
  "org": "AS16509 Amazon.com, Inc.",
  "postal": "100-0001",
  "timezone": "Asia/Tokyo"
}
```

## 🛠️ Справочник полей JSON API

| Поле | Тип | Описание | Пример значения |
| :--- | :--- | :--- | :--- |
| `ip` | String | Публичный IP-адрес клиента | `"57.182.85.71"` |
| `hostname` | String | Имя хоста (обратный DNS) | `"ec2-57-182-85-71.ap-northeast-1..."` |
| `city` | String | Город | `"Tokyo"` |
| `region` | String | Регион, область или штат | `"Tokyo"` |
| `country` | String | 2-буквенный ISO-код страны | `"JP"` |
| `loc` | String | Координаты геолокации (`широта,долгота`) | `"35.6895,139.6917"` |
| `org` | String | Номер автономной системы (ASN) и провайдер | `"AS16509 Amazon.com, Inc."` |
| `postal` | String | Почтовый индекс | `"100-0001"` |
| `timezone` | String | Часовой пояс (временная зона) | `"Asia/Tokyo"` |

## ⚙️ Разработка и локальный запуск

Проект построен как статический сайт и развертывается на **Cloudflare Pages** с использованием Edge-функций.

### Требования

*   Node.js (версии 18 и выше)
*   npm или yarn

### Установка

Клонируйте репозиторий и установите зависимости:

```bash
npm install
```

### Локальный веб-сервер для фронтенда

Для запуска сервера разработки Vite (с поддержкой hot-reload):

```bash
npm run dev
```

### Локальная симуляция функций Cloudflare Pages

Для запуска серверных API-функций (таких как `/json`, `/api/myip` и др.) на локальной машине с помощью Wrangler:

```bash
npm run pages:dev
```

Локальный сервер запустится на `http://localhost:8788`. Вы можете протестировать запросы из терминала:

```bash
curl http://localhost:8788
curl http://localhost:8788/json
```
